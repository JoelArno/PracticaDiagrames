# Sistema de Gestió Bancària - Disseny Orientat a Objectes (DOO)

Aquest repositori conté la pràctica de disseny i implementació de l'esquelet d'una aplicació bancària, seguint els requisits del mòdul de Disseny de Programari.

## 📝 Descripció del Projecte
L'objectiu principal ha estat modelar una estructura bancària complexa que inclou:
- Gestió de **Clients** i **Empleats** (amb herència de Persona).
- Organització per **Sucursals**.
- Diversitat de **Productes**: Comptes Corrents, Comptes a Termini, Fons d'Inversió i Carteres de Valors.
- Relacions d'agregació (Comptes amb Targetes) i composició (Cartera amb Valors).

## 📂 Estructura del Repositori
- `/src`: Conté el codi font Java (.java) amb la implementació de l'esquelet i documentació JavaDoc.
- `/diagrames`: Inclou els diagrames UML generats (Classes, Casos d'Ús, Seqüència, Activitat i Comunicació).

## 🚀 Tecnologies Utilitzades
- **Java**: Per a la programació de les classes.
- **Mermaid**: Per a la generació de diagrames mitjançant codi.
- **VS Code**: Com a entorn de desenvolupament integrat.
